# Dark Neon Metro

## 개요

- 공공데이터 API를 활용해 실제 지하철 위치 정볼르 3D 가상 공간에 실시간으로 동기화 하는 프로젝트

## 의도

- 평면적인 텍스트 데이터를 3D 공간에 시각화하여 직관적인 데이터 이해가 가능한 디지털 트윈 환경을 구축
- <span style="border-radius:4px; background-color:#ACACAC; color:#000; padding:0 5px;">Socket.io</span>를 활용해 서버로부터 전달받는 데이터가 화면에 즉각 반영되는 프로세스를 설계하며 실시간 통신 기술을 내재화
- 2D UI의 한계를 넘어 <span style="border-radius:4px; background-color:#ACACAC; color:#000; padding:0 5px;">Three.js</span> 기반의 3D 렌더링 기법을 학습하고 프론트엔드 개발자로서의 전문성을 강화

## Tech Stack

| Category   |               Tech                |
| :--------- | :-------------------------------: |
| Framework  |  Next.js(App Router), Typescript  |
| Library    |        Three.js, Socket.io        |
| Deployment | Vercel(Frontend), Railway(Server) |
| CI/CD      |          Github Actions           |

### 해당 프로젝트는 데스크탑 환경에 최적화되어 있음.
